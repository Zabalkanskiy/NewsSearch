# NewsApp

News Viewing Application for Android

Written in Kotlin

Uses libraries: Dagger2, Retrofit, RxJava, Coroutines, Room, mvp.moxy, Glide, swiperefreshlayout, orbit-mvi, Lottie

Implemented approaches: Clean Architecture, MVP, MVVM

***
